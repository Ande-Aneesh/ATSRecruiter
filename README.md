# 🚀 AI Resume Surgeon

### Smart Resume Analyzer & Career Mentor

---

## 🧠 Overview

AI Resume Surgeon is an AI-powered web application that analyzes resumes like both an Applicant Tracking System (ATS) and a human recruiter. It not only identifies weaknesses but actively improves resumes by rewriting bullet points, suggesting enhancements, and enabling users to download a polished version.

This project is built for **DeployIt 2026 Hackathon** to solve a critical problem faced by students — getting rejected due to poorly optimized resumes.

---

## ❗ Problem Statement

Most resumes get rejected before reaching a recruiter due to:

* ATS filtering issues
* Weak or generic bullet points
* Missing role-specific keywords
* Lack of measurable impact

Existing tools only highlight problems but **do not fix them effectively**.

---

## 💡 Our Solution

AI Resume Surgeon goes beyond analysis. It:

* Evaluates resumes using AI reasoning
* Identifies gaps and weaknesses
* Rewrites content with strong impact
* Provides career guidance
* Allows editing and downloading improved resumes

👉 It transforms resumes, not just reviews them.

---

## 🔥 Key Features

### 📊 Resume Analysis

* ATS Score (keyword + formatting analysis)
* Recruiter Score (clarity + storytelling)

### 🧠 Strengths & Weaknesses

* Deep AI reasoning with explanations

### 🎯 Keyword Gap Analysis

* Matches resume with job description
* Highlights missing and matched keywords

### ✍️ Bullet Point Rewriting

* Converts weak statements into impactful ones
* Uses action verbs + measurable results

### 💬 AI Career Mentor

* Suggests skills to learn
* Provides career path guidance

### 🛠 Resume Editor

* Editable improved resume
* Accept/reject AI suggestions

### 📄 PDF Export

* Download a clean, ATS-friendly resume

---

## 🧠 How It Works

1. User uploads resume or uses sample data
2. AI analyzes content using structured prompts
3. Generates:

   * Scores
   * Feedback
   * Rewrites
4. User edits improved resume
5. Downloads final version

---

## ⚙️ Tech Stack

* **Frontend/UI**: Lovable (No-code AI builder)
* **AI Engine**: LLM (for reasoning, rewriting, and analysis)
* **Processing**: Prompt-based structured output
* **Export**: PDF generation
